# Fitness Tracker Website

## Project Description
This website helps users track their workouts, view summaries, and stay motivated with visuals and quotes. It includes multiple pages such as Home, Media, Summaries, and About/Contact.

## Installation Instructions
1. Clone this repository:  
   `git clone https://github.com/<murenzi>/fitness-tracker-website`
2. Open `index.html` in your browser.

## Usage Instructions
- Navigate through the menu to explore each section.
- Use the contact form to send feedback or connect with the developer.
